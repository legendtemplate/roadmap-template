### Prisma 
> [Quickstart](https://www.prisma.io/docs/getting-started/quickstart) <br /> 

***

First
```bash
npm install prisma @prisma/client
```
then init
```bash
npx prisma init
```

push to database for making tables
```bash
npx prisma db push
```

show table on studio
```bash
npx prisma studio
```

generate
```bash
npx prisma generate
```
